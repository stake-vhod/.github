# .github
Stake вход доступен через актуальную рабочую ссылку, которая открывает платформу даже при блокировках основного домена. Через неё можно авторизоваться без ошибок, управлять балансом, запускать игры, использовать бонусы и обращаться в поддержку. Доступ полностью безопасен и работает на всех устройствах без VPN.
